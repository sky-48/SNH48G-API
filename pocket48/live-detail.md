# 直播详情

POST

https://plive.48.cn/livesystem/api/live/v1/getLiveOne

## 请求头

```python
{
    "version": "5.0.1",
    "os": "android",
    "Content-Type": "application/json;charset=utf-8",
}
```

## 表单

```python
{
    "type": 1,
    # "userId": 0,
    "liveId": "5b83e2b00cf2234479b4cbc9"
}
```

## 已知表单参数说明

名称 | 类型 | 说明 | 必填 | 默认值 | 备注
------- | -------- | ------- | ---- | ----- | ----
type | int | 是否回放 | Y | 0 | 1-视频/ 2-电台
userId | int |  | N | 0 | 
liveId | str | 直播id | Y |  | 

## 提示

经测试，必填信息不填返回信息：
```python
{
    "status": 20018,
    "message": "直播内容取得失败"
}
```
